Accessibility Statement

Last reviewed: April 17, 2026  ·  Standard: IS 5568 / WCAG 2.1 AA

Striving for WCAG 2.1 Level AA compliance — Israeli Standard IS 5568

Our commitment

Aviel Gendler is committed to ensuring digital accessibility for people with disabilities. I continuously work to improve the user experience for everyone and apply relevant accessibility standards, including:

  • WCAG 2.1 Level AA — Web Content Accessibility Guidelines published by the W3C
  • Israeli Standard IS 5568 — the Israeli equal rights regulation for accessibility of internet services, based on WCAG 2.1

Measures taken

To support accessibility, this site implements the following:

  • Semantic HTML with proper landmark regions (<nav>, <main>, <footer>, <section>)
  • Logical heading hierarchy (H1 → H2 → H3) on every page
  • All interactive elements are reachable and operable via keyboard (Tab, Enter, Space)
  • Visible focus indicators on all focusable elements
  • Skip-to-content link at the top of every page
  • Descriptive alt text on all informative images; decorative images marked alt=""
  • Text contrast ratios meeting or exceeding 4.5:1 for normal text and 3:1 for large text
  • ARIA labels on navigation regions, interactive controls, and modal dialogs
  • Hamburger menu with aria-expanded and aria-controls attributes
  • Mode switcher with role="group" and aria-pressed on toggle buttons
  • Animations respect the prefers-reduced-motion media query
  • Cookie consent dialog includes keyboard-operable controls and role="dialog"
  • All pages use lang="en" on the <html> element

Known limitations

The following limitations are known and are being worked on:

  • Animated marquee strip — the scrolling text banner is decorative and marked aria-hidden="true", so its content is not available to screen readers. No essential information is conveyed in the marquee.
  • Blog posts — older blog post pages use an alternative color scheme. These pages are being reviewed for full WCAG 2.1 AA compliance.
  • Logo carousel — the animated logo strip uses role="img" with a combined aria-label describing all client logos. Some users may prefer a static list; a text alternative can be provided on request.

Technical specifications

This website relies on the following technologies for conformance:

  • HTML5
  • CSS3 (including custom properties and media queries)
  • Vanilla JavaScript (ES6+)
  • WAI-ARIA 1.2

The website has been tested with:

  • Keyboard navigation (Tab, Shift+Tab, Enter, Space, Arrow keys)
  • Chrome DevTools accessibility tree
  • axe DevTools browser extension
  • Colour contrast analyzed against WCAG 2.1 Success Criterion 1.4.3

Feedback and contact

I welcome feedback on the accessibility of this site. If you experience any barriers or have suggestions for improvement, please get in touch:

Accessibility contact

Email: aviel@gendler.com

I aim to respond to accessibility-related enquiries within 5 business days.

If you are not satisfied with my response, you may contact the Israeli Equal Rights Commission or the relevant authority in your jurisdiction.

Formal complaints

If you are a user in Israel and believe this website does not meet IS 5568 requirements, you may file a complaint with the Commissioner for Equal Rights of Persons with Disabilities at the Israeli Ministry of Justice.

Date of review

This accessibility statement was last reviewed on 17 April 2026. I review and update it at least once per year or whenever significant changes are made to the site.
